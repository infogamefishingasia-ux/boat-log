Settings: Fuel Tanks

Set each boat's tank capacity and current fuel balance. The balance updates automatically as captains log voyages - adjust it here if it drifts (e.g. an external refuel not logged through the app).

Settings: Captains & PINs

Set a 4-digit PIN for each captain so they can log in on the boat app. Untick "Active" to hide a captain from the login screen without deleting their history.

Settings: Admin Logins

Set a password for each person who should be able to open this dashboard (you + your managers). Untick "Active" to disable a login without deleting it. At least one active login with a password must always remain.
